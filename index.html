📖 실제 동작을 위한 설정 가이드

1. 저장소 복사 (Fork)

위의 [GitHub 저장소 방문] 버튼을 클릭한 뒤, 오른쪽 상단의 Fork 버튼을 눌러 자신의 계정으로 가져옵니다.

2. API 키 등록

기상청 API 허브에서 발급받은 키를 Settings > Secrets > Actions에 KMA_API_KEY라는 이름으로 등록합니다.

3. 코드 수정 및 실행

위에서 생성한 지역 코드를 update_calendar.py 상단에 붙여넣고 저장하면 로봇이 매일 달력을 업데이트합니다.
